Twitter/X Read More Expander Options

Keyboard Shortcuts
You can customize the keyboard shortcuts for this extension:
Firefox Instructions:
1. Go to about:addons in your Firefox browser
2. Click the gear icon and select "Manage Extension Shortcuts"
3. Find "Twitter/X Read More Expander" in the list
4. Customize the shortcut for "Expand tweets once on current page"
Chrome Instructions:
1. Go to chrome://extensions/shortcuts in your Chrome browser
2. Find "Twitter/X Read More Expander" in the list
3. Click in the shortcut input field
4. Press the key combination you want to use
Current default shortcut: Alt+Shift+R
